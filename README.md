# 🏀 CBA 篮球信息概况

> 一站式 CBA 数据看板，覆盖球员数据、球队档案、联赛趋势与季后赛赛程。

---

## 关于本应用

**CBA 篮球信息概况**是一个面向 CBA 球迷和数据分析爱好者的轻量级 Web 应用。无需安装、无需注册，打开浏览器即可浏览中国男子篮球职业联赛（CBA）的核心数据与资讯。

应用由四个功能模块组成，覆盖从球员个人数据到联赛宏观趋势的完整信息链路。所有数据支持赛季切换、类别筛选与关键词搜索，界面采用运动竞技风格的暗色主题设计。

---

## 四大功能板块

### 🏀 球员数据看板

- 预载 **2023-24 / 2024-25 / 2025-26** 三个赛季的 TOP 15 球员数据
- 支持按**得分、篮板、助攻、抢断、盖帽**五大类别切换排行
- 跨赛季对比同一位球员的成长轨迹
- 球员 / 球队名称实时搜索过滤

### 🏟️ 球队信息概览

- 收录 **20 支 CBA 球队**的档案卡片
- 卡片展示：队名、所在城市、成立年份、总冠军次数、现任主教练
- 支持球队名称关键词搜索
- 网格布局，适配桌面与移动端

### 📊 联赛数据趋势

- 基于 **Chart.js** 渲染的三组可视化图表：
  - **赛季得分王趋势**（折线图）——追踪三个赛季得分王的场均变化
  - **本土球员 vs 外援得分对比**（柱状图）——直观呈现联赛得分结构
  - **赛季数据王雷达图**——多维度对比得分、篮板、助攻、抢断、盖帽五项核心指标
- 图表支持悬停查看详细数值

### 📅 赛程与比分

- 展示 **季后赛对阵时间线**，按时间轴排列
- 每场比赛标注对阵双方、比赛日期及比分结果
- 紧凑卡片式排版，一目了然

## 视觉设计

应用采用**运动竞技 + 赛博暗色**的视觉风格：

| 设计要素 | 说明 |
|----------|------|
| **主色调** | 暗色背景（`#0d0d1a`）+ 橙红强调（`#ff6b35`） |
| **背景纹理** | 模拟球场木地板纹路的 CSS 渐变 |
| **卡片样式** | 毛玻璃态（glassmorphism）+ 悬停上浮动画 |
| **字体** | 系统原生字体栈，中文优先 PingFang SC / Microsoft YaHei |
| **响应式** | 适配 320px 手机到 4K 屏幕，网格自动重排 |

## 技术架构

```
纯前端单页面应用
├── HTML5          — 语义化结构
├── CSS3           — 自研样式系统（CSS 变量 + 动画）
├── JavaScript     — 原生 ES6，零框架依赖
└── Chart.js 4.x   — CDN 引入，仅用于趋势图表
```

- **零后端依赖**：所有数据静态预载，无需数据库或 API
- **零构建工具**：即开即用，修改即生效
- **部署友好**：单个 `index.html` 文件，GitHub Pages / Vercel / Netlify 均可一键部署
